- Code index.html mẫu:
<html>
	   <head>
	    	<link rel="stylesheet" href="styles/default.css">
		    <script src="highlight.pack.js"></script>
		    <script>hljs.initHighlightingOnLoad();</script>
	   </head>
	   <body>
		 <pre><code class="cpp">
     
     #include <stdio.h>

int main() {
    /* my first program in C */
    printf("Hello, World! \n");
   
    return 0;
}
    </code></pre>
	  </body>
</html>

- Có thể chọn kiểu ngôn ngữ bằng thẻ <code class="cpp">
- Có thể chọn kiểu style khác nhau ví dụ như chủ đề tối, sáng trong <link rel="stylesheet" href="styles/default.css">, mặc định là default.css
